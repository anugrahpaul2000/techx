Always remember

“With Data Collection - "The Sooner , the better" is always the best answer -Marissa Mayer ”
As industrialization grows, Data Scientist will more focus on business needs, machine learning and strategic oversights, while data analysts who works for business will more focus on creating models and other stuffs. In General, Data Analysts focus more on answering to specific questions, analyzing problems and come with the better solutions, whereas Data Scientists concentrate efforts on producing insights.

In terms of technical skills, future Data Scientist will need to focus more on machine learning operation Process. Tools used for the Data Analysis are “Statistics and Data Modelling “and for the Data Science is “Hadoop, Programming Languages and Machine Learning “. Span of Domain is comparatively smaller in Data Analysis where as it is very expansive in Data Science.

Data Analytics v/s Data Analysis

Everyone must wonder that analytics and analysis are same....but no there are quite significant differences have a look into it .

Data Analytics:

  • Data analytics is general form of analytics which is used in businesses to make decisions from data which are data driven.
  • Data analytics consist of data collection and inspect in general and it has one or more usage.
  • There are many analytics tools in market but mainly R, Tableau, Python, Apache Spark, are used.
  • Data analytics lifecycle consist of Business Case Evaluation, Data Identification, Data Acquisition & Filtering, Data Validation & Cleansing, Data Aggregation. Data Analysis & Visualization, Utilization of Analysis Results.

Data Analysis

  • Data analysis is a specialized form of data analytics used in businesses to analyse data and take some insights of it.
  • Data analysis consist of defining a data,investigation, cleaning, transforming the data to give a meaningful outcome.
  • For analyse the data Open Refine, KNIME, RapidMiner, Google Fusion Tables, Tableau Public, NodeXL tools are used.
  • Sequence followed in data analysis are data gathering, data scrubbing and interpret the data precisely so that you can understand what your data want to say.
"Data Analytics" is the scientific process of transforming data into insights for better decision making. It is studying what will happen in future and exploring potential future events where as, "Data Analysis" is the process of examining, transforming and arranging raw data in a specific way to generate useful information from it. Analysis is studying what has happened in past. Ex: Why? How?

Terms we should be aware of, while going in depth of the Data Analytics:

  • Variable: It is a characteristics of any entity being studied that is capable of taking on different values.

  • Measurements: It is when a standard process is used to assign numbers to particular attributes or characteristics of variable.

  • Data: Data are recorded measurements.
    • So, suppose there is a variable you measure the phenomena, after measuring the phenomena, you are substituting some value for the variable. so the variable will take a particular value and that value is nothing but Data.
