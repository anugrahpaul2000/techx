Classification of data Analytics

Data Analytics is mainly characterized into 4 parts:

1) Descriptive Analytics (What happened?):

  • It is the conventional form of Business Intelligence and Data Analysis.
  • It seeks to provide a depiction on “Summary view” of facts and figures in an understandable format.
  • This either inform or prepare data for further analysis.
  • Descriptive analysis or statistics can summarize raw data and convert it into a form that can be easily understood by humans.
  • They can describe in detail about an event that has occurred in past.
  • A common example of descriptive analysis is company reports that simply provide a historic review like:
    • Data queries
    • Data dashboard
    • Reports
    • Descriptive statistics
    • Data Visualization.

2) Diagnostic Analytics (Why did it happened?):

  • It is a form of advance analytics which examines data or content to answer the questions “Why did it happen?”.
  • Tools aid on analysts to dig deeper into an issue so that they can arrive at the source of a problem.
  • It uses techniques such as:
    • Data Discovery
    • Data Mining
    • Correlation

3) Predictive Analytics (What will happen?):

  • It helps to forecast trends based on current events.
  • Predicting the probability of an event happening in future or estimating the accurate time, can all be determined with the help of predictive analytical model.
  • Many different but co-dependent variables are analysed to predict a trend in this type of analysis.
  • Technique used in Predictive Analysis are:
    • Linear regression
    • Time series Analysis and Forecasting
    • Data Mining

4) Prescriptive Analytics (How can we make it happen?):

  • It is the set of techniques to indicate the best course of action.
  • It tells what decision to make to optimize the outcome.
  • The goals of prescriptive analytics is to enable:
    • Quality improvement
    • Service enhancements
    • Costs reduction
    • Increasing Productivity
  • Tools used in Prescriptive analytics are:
    • Optimization Model
    • Simulation Model
    • Decision Analysis

  • Elements of Data Analytics:
